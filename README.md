# Admin-table-Ui

This is a react based UI for the admin table.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

## Features
```
1. Admin can view all the users in the system.
2. Admin can edit the user details.
3. Admin can delete the user.
```

## Technologies Used
```
1. React
2. JSX, HTML, CSS3
```
